# 可重入读写锁判断 #
1. 读锁是否可获取（包括重入和写锁降级）?

	原则：
		1. 判断是否有写锁的线程
		2. 是否持有该锁的读线程（读锁重入）
		3. 是否有写线程请求持有锁
	
	实现：
	- 是否持有锁的写线程
	- 是否持有该锁的读线程
	- 是否有写线程请求持有锁
	- 可读


2. 写锁是否可获取（包含重入和读锁升级）
	
	原则：
		1. 首先判断线程是否唯一持有读锁的线程
		2. 再判断读锁是否可获取
		3. 然判断写锁是否已被获取
		4. 最后判断是否持有该锁的写线程（写锁重入）
	
	实现：
	- 是否唯一持有锁的读线程
	- 读锁是否被获取
	- 写线程锁是否被获取
	- 是否持有锁的写线程
	- 可写
